<strong>, <b>, <em>, <i>, <u>, <mark> y <small>

strong: indica importancia fuerte.

b: pone texto en negrita visual.

em: indica enfasis en una frase.

i: suele usarse para una voz alternativa o estilo visual.

u: subraya texto.

mark: resalta una palabra.

small: muestra texto de menor tamano.

HTML

<p><strong>strong:</strong> indica importancia fuerte.</p>
<p><b>b:</b> pone texto en negrita visual.</p>
<p><em>em:</em> indica enfasis en una frase.</p>
<p><i>i:</i> suele usarse para una voz alternativa o estilo visual.</p>
<p><u>u:</u> subraya texto.</p>
<p><mark>mark:</mark> resalta una palabra.</p>
<p><small>small:</small> muestra texto de menor tamano.</p>

code,
kbd,
samp {

  /* Pinta el fondo para separar visualmente el contenido tecnico del texto comun. */
  background-color: #f1f5f9;
}

pre {
  /* Agrega espacio interno para que el bloque de codigo se lea mas comodo. */
  padding: 12px;
  /* Dibuja un borde alrededor para que el bloque se vea como una caja aparte. */

  /* Si el codigo es mas ancho que la caja, permite desplazarse en horizontal. */
  overflow-x: auto;
}
